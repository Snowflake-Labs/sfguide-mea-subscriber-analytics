# Agentic Audience Analytics with Snowflake Intelligence

## Overview

Build an end-to-end agentic AI solution for streaming-media advertising analytics using Snowflake Intelligence. This guide demonstrates how to deploy synthetic data generation pipelines, data transformation layers, semantic views for natural language queries, and five Streamlit applications for audience segmentation and subscriber analytics.

## Step-By-Step Guide

For prerequisites, environment setup, step-by-step guide and instructions, please refer to the [Snowflake Guide](https://www.snowflake.com/en/developers/guides/agentic-audience-analytics/).
